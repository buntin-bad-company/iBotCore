# iBotCore
iBotCore is an authoritative bot designed to host multiple functions, intended to operate on my server.

### Using below

- Discord.js
- Bun Buildin APIs

# Optional Modules Requipments
- Apache & Nginx Web Server
- Cloudflare DNS API
- MDA & Imap Server
- node-imap
- mailparser
- Elysia cron module
- Sqlite3 Database
